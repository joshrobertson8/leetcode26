# 680 - Valid Palindrome II

## Difficulty: Easy - valid-palindrome-ii

Given a string s, return true _if the_s _can be palindrome after deletingat most one character from it_.

Example 1:

    Input: s = "aba"
    Output: true

Example 2:

    Input: s = "abca"
    Output: true
    Explanation: You could delete the character 'c'.

Example 3:

    Input: s = "abc"
    Output: false

Constraints:

  * 1 <= s.length <= 105
  * s consists of lowercase English letters.
